## FATORIAL

---

### Início

- [x] Ler (A)
- [x] Contador = 1
- [x] Fatorial = A (armazena o somatório das multiplicações)
- [x] Enquanto (Contador < A) Faça
  - [x] Fatorial = Fatorial * (A-Contador)
  - [x] Contador = Contador + 1
- [x] Fim Enquanto
- [x] Exibir (Fatorial)
